### Getting Started

- Project setup <br/>
  `npm install`

- Compiles and hot-reloads for development <br/>
  `npm run serve`

- Compiles and minifies for production <br/>
  `npm run build`

- Lints and fixes files <br/>
  `npm run lint`

- Customize configuration <br/>
  <a href="https://cli.vuejs.org/config/">See Configuration Reference.</a>

- Step Doing <br/>
  `vue add router`
  `vue add vuetify`
